.. slide:: Manage AWS: Automation with Fabric

    This document contains no slides.

You've learned how to use Boto to create and destroy AWS resources. Now you'll
learn to automate that process using `Fabric`_.

.. _Fabric: http://docs.fabfile.org/en/latest/

Tasks
=====

Follow the tutorial on fabric (Links to an external site.).

Be sure to note that at the end you are asked to add two more fabric functions
that are not specified in the tutorial.  This is your chance to shine!

Submitting Your Work
====================

For your submission, upload your ``fabfile.py``. Also take a screenshot of your
AWS instance showing the nginx default page in your web browser. Upload that as
well.

Use the comment feature to supply any questions, comments or reflections you
have about the process you have done here.
