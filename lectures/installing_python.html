Installing Python¶

Python is in an era of transition. Version 2 of the language is being phased out (the end-of-life is set for 2020). Version 3 is growing in support and usage. However, in your professional lives as Python programmers you are still quite likely to be working in Python 2.

This class is focused on developing code that is compatible with both Python 2 and Python 3. The intent is to ensure that you can contribute to updating the code base in your future position. You’ll be equipped to help break up the log-jam of older code that needs upgrading.

You will, therefore, need to have a working version of both Python 2 and Python 3. This lecture is intended to help you achieve that goal.

For OS X¶

Python 2 comes pre-installed on OS X. However, it’s not the most up-to-date. You can use the Homebrew package manager to install a more recent version, and to install Python 3. Use that link to install the package manager, if you have not already done so.

Once you’ve got Homebrew installed, you will install each version:

$ brew install python
...
$ brew install python3
...

For Linux¶

Depending on which distribution and version of Linux you are using, the system Python may be 2 or 3. On most recent versions of most distributions you can use the system package manager to install the other version. This example assumes Ubuntu linux 16.04.

By default, the Python installed on Ubuntu 16.04 (Xenial) is version 3.5. You’ll need to update. To install the required extensions to Python 3, as well as what you’ll need to work with Python 2, issue the following command:

$ sudo add-apt-repository ppa:jonathonf/python-3.6
...
$ sudo apt-get update
...
$ sudo apt-get install python3.6 build-essential python3-dev python3-venv python python-dev

The Result¶

When you’ve completed these tasks, you should be able to use either Python 2 or Python 3.

What Versions Do I Have?¶

You can use the -V command-line flag to check the version of your Python executables without starting an interpreter:

$ python3 -V
Python 3.6.1

$ python2 -V
Python 2.7.13

$ python -V
Python 2.7.13
